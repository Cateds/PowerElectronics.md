# 往年题知识点汇总

> 统计范围：2022、2023、2024、2025 共 4 套 Power Electronics 主考试卷。这 4 套题均按当前 PDF 文本整理。统计用于判断复习优先级，不等同于官方命题概率

## 统计口径

- 频次按小问或明确子任务计数；同一综合大题可能贡献多个知识点。
- 题号格式如 `2024 Q3(a)(ii)`；若一个小问连续覆盖多个子任务，会用 `2022 Q3(a-h)` 这样的压缩写法。
- 公式页不计入统计；只统计正式题目。
- 课程链接优先指向本仓库最接近的 lecture 或 heading；个别跨章节题会链接到最相关的一组讲义。
- 优先级是复习建议：A 必须熟练，B 高频重要，C 低频但容易送分或丢分。

## 知识点明细

### A. 必须熟练掌握

| 知识点                         | 考过的题                                                                                                                  | 复习重点                                                                                                                                                       |
| ------------------------------ | ------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 二极管整流器 / PIV / 输出波形  | `2022 Q1(e)`, `2022 Q2(a-b,e)`, `2023 Q1(d)(i)`, `2023 Q1(e)`, `2024 Q1(e)`, `2024 Q2(c-f)`, `2025 Q1(e)`, `2025 Q2(a-b)` | 能识别半波、中心抽头全波、桥式整流；会画 $V_R$、二极管电压、带/不带电容的波形；熟悉 PIV 在桥式和中心抽头中的差别。                                             |
| 平滑电容 / 纹波 / 变压器额定值 | `2022 Q2(a,c-d)`, `2023 Q1(d)(i)`, `2024 Q2(a,c,e)`, `2025 Q2(a)`                                                         | 会用题干给出的 conduction angle 或 capacitor supply time 估算 ripple；知道增大电容会降低纹波但增大二极管脉冲电流和变压器 VA 压力。                             |
| SCR / 相位控制整流             | `2022 Q1(d)`, `2023 Q1(c)`, `2024 Q1(a-b)`, `2025 Q1(b)`, `2025 Q2(c)`                                                    | 记住 SCR 半可控：能触发导通，不能主动关断；会画延迟角 $\alpha$ 后开始导通的波形；会区分半波、全波和 SCR 桥平均值/RMS 公式。                                    |
| Buck 变换器 / CCM              | `2022 Q3(a-h)`, `2023 Q3(a-d)`, `2025 Q3(a)(i-vi)`                                                                        | 核心是 $V_o = D V_i$、$v_L = L\,di/dt$、稳态电感电流周期净变化为 0；会算 $D$、$t_{on}$、$I_{L,avg}$、$I_{L,max/min}$，并能画 $i_L$、$v_L$、输入/输出电流波形。 |
| 逆变器 / PWM / Shoot-through   | `2023 Q4(a-e)`, `2024 Q4(a-b)`, `2025 Q4(i-vi)`                                                                           | 会写全桥双极性/单极性 PWM 的开关条件；知道单极性 PWM 谐波更高频、更易滤除但实现更复杂；三相题要会线电压、相序、方波六状态表和死区时间。                        |
| 热管理 / 热阻                  | `2022 Q4(c)`, `2023 Q1(d)(ii)`, `2024 Q1(d)`, `2025 Q1(d)`                                                                | 热阻串联按 $\Delta T = P\sum\theta$；共用散热器时散热器承受总功耗，但每个器件的 $\theta_{JC}$、$\theta_{CS}$ 仍按自身功耗算；derating 是热阻的另一种表达。     |
| 波形平均值 / RMS / Form factor | `2022 Q1(a)`, `2023 Q1(a)`, `2023 Q2(b)`, `2025 Q1(a)`, `2025 Q2(c)(iv)`                                                  | 平均值和 RMS 都来自积分；form factor 是 $\mathrm{RMS}/\mathrm{average}$；注意题目问的是 signed average、rectified average 还是 load current average。          |

### B. 高频重要知识点

| 知识点                      | 考过的题                                                 | 复习重点                                                                                                                                                     |
| --------------------------- | -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| MOSFET / 二极管损耗         | `2023 Q1(b)`, `2023 Q2(c-f)`, `2024 Q1(c)`, `2025 Q1(c)` | 二极管总损耗常是导通损耗加反向恢复损耗；MOSFET 题常把 $R_{DS(on)}$、$t_{on}$、$t_{off}$、$f_s$、PWM duty 全部给出，让你分别算 conduction 和 switching loss。 |
| 缓冲电路 / 振铃             | `2022 Q4(a-b)`, `2024 Q3(b)`                             | 知道 snubber 的目的不是提高效率，而是限制开关应力；会用寄生 $L$ 和 $C$ 算 $f = 1/(2\pi\sqrt{LC})$；RC/RL 缓冲器分别对应关断电压斜率和导通电流斜率。          |
| Buck-Boost / Flyback / 隔离 | `2023 Q3(e)`, `2024 Q3(a)`, `2025 Q3(b)`                 | Buck-Boost 可升降压但不隔离；需要电气隔离时选 Flyback；Flyback 输出关系比 Buck-Boost 多了匝比 $N_2/N_1$。                                                    |
| 线性稳压电源                | `2022 Q3(i)`, `2024 Q2(a-b,e)`                           | 线性稳压器效率近似 $V_o/V_i$；最大功耗通常发生在输入最高、输出电流最大时；变压器 VA 题要结合整流电流 form factor。                                           |
| 器件特性 / SOA              | `2022 Q1(b-c)`, `2024 Q1(c)`, `2025 Q1(c)`               | SOA 是器件安全工作区；MOSFET 优点常见是高输入阻抗、开关快、易驱动，缺点包括导通电阻随耐压上升、静电敏感、过压过热风险。                                      |

### C. 低频但容易送分

| 知识点                    | 考过的题                   | 复习重点                                                                                                              |
| ------------------------- | -------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| IGBT FBSOA                | `2022 Q1(b)`               | 会解释 SOA，并能把图中的电流、电压、功率、二次击穿或脉冲限制等边界说成“安全工作限制”。                                |
| 二极管瞬态参数标注        | `2022 Q1(c)`               | $V_F$、$I_F$、$V_{FP}$、$V_{rr}$、$I_{rr}$ 主要考图上标注和物理意义，别把 forward recovery 和 reverse recovery 混掉。 |
| 三相逆变器相序 / 六状态表 | `2023 Q4(d)`, `2024 Q4(a)` | 改变相序通常交换任意两相控制信号；方波六状态表要记住每次只改变一个桥臂状态。                                          |
| 桥式整流优缺点            | `2022 Q2(e)`, `2024 Q2(f)` | 桥式不需要中心抽头、变压器利用率好、PIV 要求较低；缺点是每次导通有两个二极管压降。                                    |

## 复习优先级建议

### 1. 必须会算

| 计算类知识点                    | 最低要求                                                                                                                               |
| ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| 平均值 / RMS / Form factor      | 能对分段线性波形、整流波形和电流波形积分；会把周期、峰值、绝对值和有效值分清楚。                                                       |
| 整流器 ripple / PIV / capacitor | 会根据半波/全波频率、负载电流、允许纹波、conduction angle 或供电时间求电容；会判断 PIV 和电容额定电压。                                |
| SCR 相控整流                    | 会用延迟角 $\alpha$ 写积分上下限；会算半波和全波/SCR 桥平均输出；RMS 公式要能从积分推出。                                              |
| Buck 变换器                     | 会从 $V_o = D V_i$ 求 duty；会用 $v_L = L\,di/dt$ 求电感电流上升/下降量和最大/最小值；会画关键波形。                                   |
| Buck-Boost / Flyback            | 会写 $V_o = -D/(1-D) V_i$ 和 Flyback 的匝比修正；知道隔离需求直接指向 Flyback。                                                        |
| MOSFET / diode losses           | 会分别算 conduction loss、switching loss、reverse recovery loss；知道 halving switching frequency 只明显影响 switching/recovery loss。 |
| Thermal resistance              | 会画热路并按功耗乘热阻求温升；能反推所需 $\theta_{SA}$；共用散热器要用总功耗算散热器温升。                                             |
| Snubber / ringing               | 会用 $f = 1/(2\pi\sqrt{LC})$ 算振铃频率；会从电感电压关系估算 $di/dt$ 限制和缓冲电阻功耗。                                             |
| Linear regulator supply         | 会处理 dropout、最小 secondary voltage、最坏输入下 regulator dissipation、transformer VA rating。                                      |

### 2. 必须会画和辨析

| 概念/画图类知识点 | 易错点                                                                                                |
| ----------------- | ----------------------------------------------------------------------------------------------------- |
| 整流器波形        | 带电容时负载电压近似保持在峰值附近并带小纹波；不带电容时才是半波/全波脉动波形。                       |
| 二极管电压波形    | 导通时二极管压降近似 $V_F$，截止时承受反向电压；PIV 看的是最极端的反向峰值。                          |
| SCR 相位控制      | 延迟角从每个半周期开始处量起；SCR 触发后锁定到电流过零，不是 gate signal 消失就关断。                 |
| Buck 波形         | 开关闭合时电感电流上升、断开时下降；连续模式下 $I_{L,min} > 0$，边界条件是 $I_{L,min} = 0$。          |
| PWM 全桥逆变器    | 双极性输出在 $+V_{dc}$ 和 $-V_{dc}$ 间切换；单极性每个半周在 $0$ 和一个极性之间切换，谐波更容易滤除。 |
| 三相逆变器        | 三个控制信号相差 $120^\circ$；线电压是两相输出之差；改相序通常交换两相。                              |
| Shoot-through     | 同一桥臂上下管不能同时导通；死区时间是保护措施，不是调制目标。                                        |
| Thermal circuit   | 热阻图方向从 junction 到 ambient；温度差类似电压，功耗类似电流。                                      |
| Snubber placement | RC/RL 缓冲器位置和极性影响功能；不要把所有 snubber 都简单理解成“并一个电容”。                         |

### 3. 特别注意的口径

| 口径                   | 注意事项                                                                                        |
| ---------------------- | ----------------------------------------------------------------------------------------------- |
| 峰值、RMS 与输入表达式 | 题干写 $V = A\sin(\omega t)$ 时 $A$ 是峰值；写 $80\,\mathrm{V_{rms}}$ 时峰值要乘 $\sqrt{2}$。   |
| Form factor 的 average | 题目若要求整流后的 form factor，average 通常是整流平均值，不是正负抵消后的 0。                  |
| 全波频率               | 全波整流后纹波频率是输入交流频率的 2 倍；半波仍是输入频率。                                     |
| 桥式与中心抽头 PIV     | 桥式整流每次两个二极管导通但单只 PIV 较低；中心抽头每次一个二极管导通但 PIV 往往更高。          |
| 二极管压降             | 题干会明确 ideal、$0.7\,\mathrm{V}$ 或 $1\,\mathrm{V}$；画波形和算峰值电流时别漏掉压降。        |
| SCR 平均值公式         | 半波、全波、SCR bridge 的积分系数不同；角度代入三角函数时注意使用 radians 还是 degrees。        |
| Boundary CCM           | “continuous mode at boundary condition” 通常表示电感电流刚好降到 0。                            |
| 线性稳压最坏工况       | Regulator 最坏功耗通常不是最低输入，而是最高输入、最大负载电流。                                |
| Derating               | $\mathrm{W}/^\circ\mathrm{C}$ derating 可看作热阻倒数相关的规格；不要和允许耗散功率直接相加。   |
| $m_a \le 1$            | 逆变器 PWM 在线性调制区时，基波幅值与 $m_a$ 近似线性；超过后进入 overmodulation，不按线性公式。 |

### 4. 常用作题干条件的

有些公式、条件或图表会直接出现在题干里。它们不是“不用学”，而是要求你能把题干条件匹配到正确模型并完成计算、画图或解释。

| 知识点       | 题干可能直接给出的内容                                                                 | 仍需自己掌握的部分                                                                  |
| ------------ | -------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| 整流器       | 拓扑图、输入峰值或 RMS、二极管是否 ideal、负载电阻、电容、conduction angle、PIV 要求。 | 识别半波/中心抽头/桥式；画负载和二极管电压；求 PIV、ripple、电容和额定值。          |
| SCR          | 延迟角、纯电阻负载、gate pulse、是否半波或全波、是否 DC 场景。                         | 知道 SCR 锁存和自然换流；写积分上下限；解释半可控和 DC 不适用。                     |
| Buck / DC-DC | $V_{in}$、$V_o$、$f_s$、$L$、$R$、输出功率、CCM 或 boundary condition。                | 从伏秒平衡推导输入输出关系；计算 duty、$t_{on}$、$I_{L,avg}$、纹波和最大/最小电流。 |
| Flyback      | 输入范围、输出电压、隔离需求、匝比或可调 duty。                                        | 说明为什么选 Flyback；用 duty 和匝比维持输出电压；和 Buck-Boost 区分。              |
| Inverter PWM | $v_{control}$、$v_{tri}$、$m_a$、$m_f$、三相相位差、开关状态表。                       | 写开关导通条件；区分双极性/单极性；解释谐波、相序和 shoot-through 防护。            |
| Thermal      | 功耗、ambient、$T_{j,max}$、$\theta_{JC}$、$\theta_{CS}$、$\theta_{SA}$、derating。    | 画串联热路；共用散热器时正确分配功耗；反推所需散热器热阻。                          |
| Losses       | $V_F$、$I_F$、duty、$Q_{RR}$、反向电压、$R_{DS(on)}$、$t_{on}/t_{off}$、$f_s$。        | 分别计算导通、反向恢复和开关损耗；解释频率变化影响哪一部分损耗。                    |
| Snubber      | 开关电流、供电电压、缓冲电阻、寄生电感/电容、需要限制 $di/dt$ 或振铃。                 | 判断用 RC 还是 RL 缓冲；会算振铃频率和功耗；能说明 snubber 的取舍。                 |
